Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web.

HTML Editor

Dokumen HTML dapat dibuat dengan editor teks sederhana atau editor HTML yang lebih canggih.

HTML Dasar

Struktur dasar HTML terdiri dari elemen-elemen utama seperti doctype, html, head, dan body.

HTML Elemen

Elemen HTML adalah komponen penyusun halaman web.

HTML Atribut

Atribut digunakan untuk memberikan informasi tambahan pada elemen HTML.

HTML Classes

Class digunakan untuk memberi nama pada elemen dan dapat digunakan dalam CSS dan JavaScript.

HTML ID

ID digunakan untuk memberikan identitas unik pada elemen.

HTML Iframes

Iframes digunakan untuk menyematkan halaman web di dalam halaman lain.

HTML JavaScript

JavaScript dapat digunakan dalam HTML untuk menambahkan interaktivitas.

File Path

File path digunakan untuk menentukan lokasi file dalam HTML.

Head

Bagian head berisi metadata seperti title, meta tags, dan link ke CSS.

Layout

HTML dan CSS digunakan untuk mengatur tata letak halaman web.

Responsive

Desain responsif membuat halaman web tampil baik di berbagai perangkat.

Computer Code

Tag seperti <code>, <pre>, dan <samp> digunakan untuk menampilkan kode komputer.

Semantics

HTML5 menyediakan elemen semantik seperti <article>, <section>, <nav> untuk struktur yang lebih bermakna.

Style Guide

Pedoman untuk menulis HTML yang bersih dan rapi.

Entities

HTML entities digunakan untuk menampilkan karakter khusus seperti &lt; dan &gt;.

Symbol

Simbol dalam HTML seperti &copy;, &trade;, &hearts;.

Emojis

HTML mendukung berbagai emoji menggunakan Unicode seperti 😀 (ðŸ˜€).

Charset

Meta charset menentukan jenis karakter yang digunakan dalam halaman web, seperti UTF-8.

URL Encode

URL encoding menggantikan karakter khusus dalam URL dengan kode yang sesuai.
